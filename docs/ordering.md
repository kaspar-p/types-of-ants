# Ordering guide for a new typesofants server

## Prerequisites

- Network topology: make sure you have enough switch ports!

## Supplies

- Server, if not: <https://www.amazon.ca/dp/B074P6BNGZ>
- Ethernet cable, if not: <https://www.amazon.ca/dp/B08QTZFHRY>
- Power brick! Preferably one with multiple USB 2.0 slots.
- Power cable, if not: <https://www.amazon.ca/dp/B07232M876>
- SD card for storage, if not: <https://www.amazon.ca/dp/B0CPDH6YVR>
